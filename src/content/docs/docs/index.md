Notes I want to keep and come back to — setup steps, things I worked out once
and don't want to work out again, half-shaped ideas.

Every folder under `src/content/docs/docs/` becomes a section in the sidebar.
Add a folder, get a section.

Editable two ways:

- **From an editor** — write markdown in `src/content/docs/docs/`, commit, push.
- **From the CMS** — open `/admin`, pick **Second Brain**, write, save. It
  commits to `main` and the site rebuilds.

Both paths write the same files. Git is the source of truth.

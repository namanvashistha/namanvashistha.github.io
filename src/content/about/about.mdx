Hey! I'm Naman. I build scalable backend systems and have a deep fascination with distributed architectures.

Currently, I'm working on **LimeDB**, a distributed key-value store built from scratch in Go. It started as a learning project into consistent hashing, replication, and a web-based terminal UI for monitoring.

My expertise spans **Python, Go, C/C++, JavaScript, and PHP**, with hands-on experience in payment systems, real-time applications, and DevOps automation. I've built everything from chess engines using bitboard representations to custom programming languages.

I love diving deep into system internals, optimizing performance, and solving complex architectural problems. When I'm not coding, you'll find me writing on [Dev.to](https://dev.to/namanvashistha), solving problems on LeetCode, or exploring new tech.

I believe in learning by building — most of my projects are open source on [GitHub](https://github.com/namanvashistha). Feel free to check them out, and let's connect if you want to chat about tech!
